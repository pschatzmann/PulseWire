Detailed Description

Header-only Arduino class for Manchester IR encoding.

This class encodes data into Manchester format for IR transmission. It works on both ESP32 and RP2040 platforms using the Arduino API.

Features:

  • Manchester encoding of arbitrary byte data
  • Preamble and checksum support for frame integrity
  • Header-only implementation for easy integration
  • Supports dynamic frame sizes
