Detailed Description

template<typename Preamble, typename Codec, uint32_t FREQ_HZ, size_t FRAME_SIZE, typename Signal = DigitalSignal, typename RxDriver = RxDriverGeneric, typename TxDriver = TxDriverGeneric>
class ir_transceiver::TransceiverTemplate< Preamble, Codec, FREQ_HZ, FRAME_SIZE, Signal, RxDriver, TxDriver >

Generic IR transceiver template for protocol-specific instantiation.

Template parameters:

  • Preamble: protocol-specific preamble class
  • Codec: protocol-specific codec class
  • Signal: signal implementation (default: DigitalSignal)

Usage example: using IRTransceiverNEC = TransceiverTemplate<NECPreamble, PulseDistanceCodec, DigitalSignal>; using IRTransceiverSony = TransceiverTemplate<SonyPreamble, PulseDistanceCodec, DigitalSignal>;
