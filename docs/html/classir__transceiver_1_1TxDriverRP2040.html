Detailed Description

High-performance RP2040 IR TX driver using PIO and DMA for edge-based IR transmission.

  • Uses a PIO state machine to transmit IR signal edges based on OutputEdge vectors.
  • DMA is used for efficient edge transfer to PIO, enabling non-blocking transmission.
  • TX is non-blocking: send returns immediately after starting DMA; only blocks if a new send is requested while previous DMA is running.
  • Allows concurrent IR receive and transmit on the same microcontroller.
  • Compatible with modular Codec/Preamble architecture.
Note
Supports sending and receiving on the same microcontroller without conflicts.

Start DMA transfer to send IR edges via PIO.

  • Prepares edge buffer and starts DMA transfer to PIO TX FIFO.
  • Returns immediately after starting DMA.
  • If a previous DMA transfer is still running, blocks until it completes before starting new send.
  • Allows concurrent RX and TX operation.
